# 六色思考帽マルチエージェント分析

エドワード・デボノの「六色思考帽」フレームワークを使い、6つのエージェントを**並列実行**して多角的に分析します。

## When to Use This Skill

Use this skill when the user:

- Asks to analyze a topic from multiple perspectives
- Says "六色思考帽で分析して" or "多角的に分析して"
- Wants structured pros/cons analysis with creative and emotional dimensions
- Needs a comprehensive report for decision-making
- Says "〇〇のメリット・デメリットを整理して"

## How to Execute

**必ず1回のメッセージで以下6つのエージェントを並列起動すること（逐次実行禁止）。**

各エージェントには以下の役割を与えて Task ツールで同時起動し、全結果が揃ってから統合レポートを出力する。

### 6エージェントの役割

**Agent 1 — ⚪ 白帽子（事実・データ）**
- 客観的な事実・統計・データ・市場規模・調査レポートのみを収集
- 感情・意見・推測は一切含めない
- WebSearchで最新データを取得し、出典を明記する
- 数値・パーセント・予測データを重視

**Agent 2 — 🔴 赤帽子（感情・直感）**
- 各ステークホルダーの感情・直感・本音を表現
- 論理的根拠は不要。感情語を使う
- 主要ステークホルダーそれぞれの感情を描写

**Agent 3 — ⚫ 黒帽子（リスク・問題点）**
- 批判的思考で潜在的リスク・問題・失敗要因を徹底的に列挙
- 技術・セキュリティ・組織・コスト・倫理・品質の観点から掘り下げる
- 最悪シナリオを検討する

**Agent 4 — 🟡 黄帽子（メリット・チャンス）**
- 肯定的・楽観的な視点でメリット・機会・価値を発見
- 定量的効果（〜%削減、〜倍速など）を可能な範囲で示す
- ベストシナリオと成功要因を具体的に描く

**Agent 5 — 🟢 緑帽子（創造的アイデア）**
- 既存の枠を超えた斬新なアイデア・応用・可能性を創出
- 具体的なコンセプトを命名して提案
- 5つ以上のユニークなアイデアを箇条書きで

**Agent 6 — 🔵 青帽子（統合・推奨アクション）**
- 他の5つの帽子の分析を統合し、実行可能な推奨アクションを導出
- 優先度付きのアクションリスト（TOP 3〜5）を作成
- 短期・中期・長期のロードマップを示す
- 意思決定者へのエグゼクティブサマリーを冒頭に記述

## Output Format

全エージェントの結果が揃ったら、以下の形式で統合レポートを出力する：

```
# 六色思考帽分析レポート
**テーマ**: [テーマ名]
**実施日**: [日付]

## エグゼクティブサマリー
（青帽子より・3〜5行）

---
## ⚪ 白帽子：事実とデータ
## 🔴 赤帽子：感情と直感
## ⚫ 黒帽子：リスクと問題点
## 🟡 黄帽子：メリットとチャンス
## 🟢 緑帽子：創造的アイデア
## 🔵 青帽子：推奨アクション
---
## 総合判定
[Go / Conditional Go / No Go] + 一言理由
```

## Notes

- 6エージェントは必ず**同時並列**で起動する
- 各エージェントは他のエージェントの結果を見ずに独立して分析する
- 日本語で出力する
- テーマが不明確な場合は分析前にユーザーに確認する
