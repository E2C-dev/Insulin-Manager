## 理想のLP構成（設計案）

### セクション1: ファーストビュー（Hero）
**目的**: 3秒で「これは自分のためのアプリだ」と感じさせる

```
[ヘッドライン]
インスリンの記録を、もっとかんたんに。

[サブヘッドライン]
妊娠糖尿病・1型・2型糖尿病のインスリン療法をサポートする
記録管理アプリです。

[CTA ボタン]
無料で始める（クレジットカード不要）

[ソーシャルプルーフ]
XXX人が使っています / ★★★★★ 口コミ4.8
```

---

### セクション2: 課題提起（Problem）
**目的**: ターゲットの「あるある」で共感を得る

```
こんな悩みはありませんか？

📓 手書きの記録ノートが面倒
📊 エクセルで管理しているが続かない
💊 インスリン量の計算が毎回不安
🏥 診察時に医師へうまく記録を見せられない
```

---

### セクション3: 解決策（Solution）
**目的**: インスリアが課題をどう解決するかを示す

```
インスリアなら、かんたんに解決できます。

[機能① 記録がワンタップ]
血糖値・インスリン量・食事を一画面で記録

[機能② 自動計算ルール]
「血糖値がXXならインスリンXX単位」をルール登録するだけ

[機能③ 医師と共有]
PDF出力・URL共有で診察がスムーズに
```

---

### セクション4: 開発背景ストーリー（Story）
**目的**: ブランドへの信頼・共感を高める

```
インスリアは、妊娠糖尿病を経験したひとりの女性の声から生まれました。

「血糖値を記録するアプリはあっても、インスリン量と一緒に管理できるものがなかった」

その一言が、開発のきっかけです。
```

---

### セクション5: セグメント別ユースケース（UseCase）
**目的**: 自分ごとに感じさせる

```
[タブ切り替え]

妊娠糖尿病の方へ  |  1型糖尿病の方へ  |  2型糖尿病の方へ

[妊娠糖尿病タブ]
妊娠中の血糖管理、産後のフォローまでサポート
→ 使い方詳細
```

---

### セクション6: 機能一覧（Features）
**目的**: 具体的な機能を網羅的に見せる

---

### セクション7: プラン比較（Pricing）
**目的**: 有料転換を促す

```
無料プラン  |  スタンダード（480円/月）  |  プレミアム（980円/月）
```

---

### セクション8: ユーザーの声（Testimonials）
**目的**: 社会的証明で安心感を高める

---

### セクション9: CTA（Call to Action）
**目的**: 最後の一押し

```
今すぐ無料で始める
（いつでもキャンセル可能）
```
