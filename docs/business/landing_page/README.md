# ランディングページ（LP）戦略

> インスリアのLPに関する設計方針・KPI・改善ロードマップ

---

## LPの目的

**無料登録コンバージョン率 15%** を達成すること（Year 1目標）

現在のLPは基本構成が整っているが、Phase 1（3〜5月）に向けて以下の改善が必要：

1. ファーストビューの刷新
2. 開発背景ストーリーの追加
3. セグメント別ユースケースの追加
4. ユーザー体験談の追加
5. プラン比較テーブルの見直し

---

## LP改善優先順位

| 優先度 | 改善項目 | フェーズ | 担当 |
|-------|---------|---------|------|
| ★★★ | ファーストビュー刷新 | Phase 1 | 開発 |
| ★★★ | 開発背景ストーリー追加 | Phase 1 | コピー |
| ★★ | セグメント別ユースケース | Phase 1 | コピー + 開発 |
| ★★ | ユーザー体験談セクション | Phase 1 | コピー |
| ★★ | プラン比較テーブル | Phase 2 | 開発 |
| ★ | SEO最適化（メタ・構造） | Phase 1〜2 | 開発 |

---

## KPI

| 指標 | 現状 | Phase 1末目標 | Year 1末目標 |
|-----|------|-------------|------------|
| LPコンバージョン率 | 未計測 | 10% | 15% |
| 直帰率 | 未計測 | 60%以下 | 50%以下 |
| 平均滞在時間 | 未計測 | 1分30秒以上 | 2分以上 |
