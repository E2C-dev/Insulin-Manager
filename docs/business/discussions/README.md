# ディスカッションハブ

> 意思決定・議論・アイデアを記録・管理するスペース

---

## このディレクトリの使い方

### 新しいディスカッションを始める

1. `discussions/YYYY-MM/` フォルダに新ファイルを作成
2. ファイル名: `TOPIC_NAME.md`（例: `lp_hero_copy.md`）
3. テンプレート: [templates/discussion_template.md](templates/discussion_template.md) を使用

### ディスカッションの状態管理

ファイル冒頭に以下のステータスを明記する：

```
ステータス: 🔵 議論中 / 🟡 決定待ち / ✅ 決定済み / ⏸ 保留
```

## エージェントへのガイド

- **議論のたたき台を出してほしい** → テンプレートを使って草案を作成し、`🔵 議論中` で保存
- **決定事項を記録してほしい** → 既存ファイルを `✅ 決定済み` に更新し、各ドキュメントの該当箇所にも転記
- **過去の議論を参照したい** → 月別フォルダを検索
