# インスリア（Insulia）Go-to-Market（GTM）戦略

> 策定日: 2026年2月22日
> バージョン: 1.0

---

## 1. ターゲットセグメント全体像

インスリア のターゲットは「インスリン療法中の糖尿病患者」であるが、セグメントによって課題・情報収集行動・意思決定プロセスが大きく異なる。セグメント別に獲得戦略を設計する。

| セグメント | 日本国内患者数（推計） | インスリン使用率 | 優先度 |
|-----------|-------------------|--------------|--------|
| 妊娠糖尿病 | 年間約6万人 | 約25〜30% | ★★★ 最優先 |
| 1型糖尿病 | 約12万人 | ほぼ100% | ★★★ 最優先 |
| 2型糖尿病（インスリン療法中） | 約230万人 | 約20〜25% | ★★ 中期優先 |
| その他（膵性糖尿病等） | 約10万人 | 変動 | ★ 長期 |

**Phase 1優先セグメント**: 妊娠糖尿病・1型糖尿病
理由: サービスの開発背景と親和性が高く、口コミ伝播率が高いコミュニティが存在する。

## 2. セグメント別獲得戦略

### 2-1. 妊娠糖尿病セグメント

#### ターゲットプロフィール
- 属性: 妊娠中〜産後1年以内の女性、20〜40代
- 課題: インスリン投与量の自己管理に不安、記録の手間、食事と血糖値の相関把握
- 情報源: Instagram、X（旧Twitter）、妊娠アプリ（トツキトオカ等）、産婦人科
- 意思決定: 感情的・共感的（「同じ経験をした人の声」を重視）

#### 獲得チャネル

**Instagram（最重要）**
- ターゲット: 妊娠糖尿病タグをフォロー中のユーザー
- 訴求軸: 「妊娠糖尿病の管理、もっとラクに」「血糖値記録がアプリ1つで完結」
- コンテンツ形式: リール（体験談・使い方動画）、カルーセル（食事×血糖値の実例）
- ハッシュタグ: #妊娠糖尿病 #妊娠糖尿病管理 #妊娠糖尿病ごはん #インスリン #マタニティライフ
- 推奨アクション: 患者体験談をもとにしたリール投稿（週2本）

**X（旧Twitter）**
- ターゲット: 「妊娠糖尿病」「インスリン 自己注射」でつぶやくユーザー
- 訴求軸: 日常の共感投稿 → アプリ紹介
- 運用方針: 公式アカウントでの共感コメント・RT、患者コミュニティへの自然な参加
- ハッシュタグ: #妊娠糖尿病 #GDM #妊娠糖尿病ママ

**コミュニティ連携**
- 妊娠糖尿病サポートグループ（Facebook・LINE等）へのアプリ紹介
- 産婦人科・助産院への資料配布（医師向け一枚資料の作成）

**インフルエンサー戦略**
- マイクロインフルエンサー（フォロワー1,000〜10,000人）の妊娠糖尿病経験者に無料プレミアム提供
- 体験レポート投稿を依頼（PR表記あり）
- 目標: Phase 1で10名の体験レポーター確保

### 2-2. 1型糖尿病セグメント

#### ターゲットプロフィール
- 属性: 10代〜50代、生涯にわたってインスリン療法が必須
- 課題: カーボカウント・補正投与の計算負担、長期的な記録管理、CGMとの連携
- 情報源: 1型糖尿病専門コミュニティ、Twitter、患者会、専門医
- 意思決定: 論理的・機能重視（正確な数値・細かい設定への要求が高い）

#### 獲得チャネル

**専門コミュニティへの参入**
- Type1Riders（スポーツ×1型糖尿病コミュニティ）へのアプリ紹介
- 「1型糖尿病仲間」等のSNSグループへのアプローチ
- 日本IDDMネットワークへの認知活動

**X（旧Twitter）**
- ターゲット: #1型糖尿病 #T1D #TypeOneDiabetes タグのアクティブユーザー
- コンテンツ: カーボカウント計算のヒント、CGM連携情報、機能アップデート告知
- 1型糖尿病インフルエンサーとの連携（医師・患者ブロガー）

**専門家・医療機関連携**
- 糖尿病専門医・糖尿病療養指導士（CDEJ）へのアプリ紹介
- 学会（日本糖尿病学会等）での認知活動（中期施策）

**SEO・ブログコンテンツ**
- 「カーボカウント 計算 アプリ」「1型糖尿病 管理 スマートフォン」等のロングテールKW
- 1型糖尿病患者向けの詳細ガイド記事の作成

---

### 2-3. 2型糖尿病（インスリン療法中）セグメント

#### ターゲットプロフィール
- 属性: 40〜70代が中心、インスリン療法移行後の患者
- 課題: 記録の継続困難（デジタルリテラシーの差）、投与量の判断への不安
- 情報源: 主治医・看護師、健康情報サイト（NHK健康・ためしてガッテン系）、家族
- 意思決定: 医師推奨が最大の信頼要素

#### 獲得チャネル

**医療機関連携（メインチャネル）**
- 内科・糖尿病内科クリニックへの営業資料配布
- 糖尿病療養指導士（CDEJ）・看護師へのアプリ紹介
- 「先生に薦められたアプリ」という信頼形成が鍵

**健康情報サイト連携**
- SEO：「糖尿病 血糖値 記録 アプリ」「インスリン 記録 スマホ」
- 健康メディア（マイナビ健康・ヘルスケア大学等）への記事寄稿・PR

**薬局連携**
- 調剤薬局でのアプリ紹介カード配布（Phase 2以降）

## 4. 獲得ファネル設計

```
認知（Awareness）
  ├─ Instagram/X投稿 → プロフィールへ流入
  ├─ SEO記事 → ランディングページへ流入
  └─ 口コミ・紹介 → 直接訪問

  ↓

興味関心（Interest）
  ├─ LP（ランディングページ）での機能紹介
  ├─ 体験談・スクリーンショットの閲覧
  └─ プラン比較ページ

  ↓

無料登録（Activation）
  ├─ アプリダウンロード
  └─ 初期設定（インスリンプリセット・計算ルール設定）

  ↓

無料→有料転換（Conversion）
  ├─ 機能制限への到達（Phase 2以降）
  ├─ アップグレード訴求プッシュ通知
  └─ 30日無料トライアル（プレミアム）

  ↓

継続・推奨（Retention & Referral）
  ├─ 記録継続のリマインダー
  ├─ 「友達を招待」機能（紹介特典）
  └─ SNSでの自発的な体験シェア
```
